## ✅ Completed Tasks

### Task 1 — Hangman Game
A text-based Hangman game where the player guesses a secret word one letter at a time.

**How to Run:**
```bash
python hangman_game.py
```

**Features:**
- Random word selection from predefined list
- ASCII art hangman figure
- Max 6 wrong guesses allowed
- Win/lose detection

**Concepts Used:** `random`, `while loop`, `if-else`, `strings`, `lists`

---

### Task 2 — Stock Portfolio Tracker
A simple stock tracker that calculates total investment based on manually defined stock prices.

**How to Run:**
```bash
python stock_portfolio_tracker.py
```

**Features:**
- Hardcoded stock price dictionary (AAPL, TSLA, GOOGL, MSFT, AMZN, META)
- User inputs stock name and quantity
- Calculates total investment value
- Optional save to `.txt` or `.csv` file

**Concepts Used:** `dictionary`, `input/output`, `basic arithmetic`, `file handling`
